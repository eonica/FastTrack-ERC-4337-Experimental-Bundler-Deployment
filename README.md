# Experimental ERC-4337 Bundler Setup - Fast Track

The following steps 

## Creating the directory structure

Create the following directory structure on the server (host machine):

/opt/powerexp/anvil
/opt/powerexp/alto
/opt/powerexp/state
/opt/powerexp/results

## State setup

Copy state.json in the state subfolder. Make a state.bak copy for backup. 

## Scripts setup

Copy the AnvilDockerfile in the anvil subfolder and the AltoDockerfile in the alto subfolder.
